# FC-simulator
> 用go实现一个NES/FC/红白机模拟器
### GUI：
选择了 fyne，处理ui的代码较少，性能一般，胜在简单。
### 音效
支持音效
### 支持情况
已支持mapper0/2/3/4的游戏，如冒险岛/沙曼陀蛇/魂斗罗/超级马里奥等大部分游戏
### 桌面版使用方式
go run main.go xxx.nes
### 桌面版操作
```
手柄1:
W/S/A/D 上下左右
F/G A/B键
R/T 选择/暂停

手柄2:
方向键 上下左右
U/I A/B键
J/K A/B键

系统按键：
Q 重启游戏
```
