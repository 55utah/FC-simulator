# FC-simulator
> 用go实现一个NES/FC/红白机模拟器
### GUI：
选择了 fyne，处理ui的代码较少，性能一般，胜在简单。
### 支持情况
已支持mapper0/2/3的游戏，如冒险岛/沙曼陀蛇/魂斗罗/超级马里奥等
### 桌面版使用方式
go run main.go xxx.nes
### 桌面版操作
W/A/S/D 方向键
U/I 选择/暂停
J/K A/B键
